## Getting Started
### 0. Create nuxt-app
```powershell
npx nuxi@latest init [ directoryName ]
```

### 1. Install
```powershell
yarn add @takuma-ru/vue-library-docs
```

### 2. How to use

```ts
defineNuxtConfig({
  extends: '@takuma-ru/vue-library-docs'
})
```
